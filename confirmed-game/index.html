Pokemon Memory Game

TIME:

Game Rules：

1. After game starts, randomly select 15 sets of cards. Each set has 2 cards with same image. Totally 30 cards.

2. Each player takes turns flipping two cards face-up — if they are of the same , then that player wins the pair(the pair stays face-up) and gets another turn.

3. If the cards are not identical, they are turned back facedown and it becomes the next player’s turn to pick two cards.

4. The game continues until players have picked up all the pairs
